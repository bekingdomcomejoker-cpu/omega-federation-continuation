## TROUBLESHOOTING

### Issue: "Meta AI not showing"

**Solution:**
1. Refresh the page (Ctrl+R or Cmd+R)
2. Clear browser cache
3. Log out and log back in
4. Try a different browser

### Issue: "Message not sending"

**Solution:**
1. Check internet connection
2. Try refreshing the page
3. Clear browser cache
4. Try a different browser

### Issue: "Context seems lost"

**Solution:**
1. Use the Restart Prompt
2. Paste it as a new message
3. Meta AI will re-establish context

### Issue: "Getting policy warnings"

**Solution:**
1. Use /gap to document the policy conflict
2. Meta AI will acknowledge the gap
3. Continue with alternative approach
